Introduction

Work in progress. Likely to change in future releases.

This reference defines the RDF vocabulary for the machine-readable syntax of the Datalog-like language supported by the Oroboro Java RDF processing framework.

Prefixes

http://oroboro.net/schema/datalog#
owl http://www.w3.org/2002/07/owl#
rdf http://www.w3.org/1999/02/22-rdf-syntax-ns#
rdfs http://www.w3.org/2000/01/rdf-schema#
