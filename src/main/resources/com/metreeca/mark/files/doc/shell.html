oroboro

oroboro is a command-line tool for executing Datalog programs.

Usage

oroboro [[options] [origin] ...]

If one or more RDF origins are specified, decodes them using a shared catalog for URI management and encodes the resulting graph to the standard output. Input origins are specified as URIs, resolved against the file: URI for the current working directory. A single hyphen ('-') identifies the standard input.

If no origin is specified, enters the command loop of an interactive Datalog interpreter.

Options

--nt, --ntriples
--ttl, --turtle
--rdf, --rdfxml
Configures the fallback RDF serialization format for handling origins and targets whose format cannot be guessed from the filename extension of their URI.
--owl origin
Classify the working graph using the OWL RL/RDF reasoner under the control of the OWL ontology decoded from origin and encodes the results to the standard output. If specified more than once, merges ontologies before applying them.
--dl origin, --datalog origin
Updates the working graph under the control of the Datalog program decoded from origin and encodes the results to the standard output. If specified more than once, merges programs before executing them.

Directives

Directives are either included in programs or executed interactively from the shell command loop. References are interpreted as a relative URI and resolved against the current base of the shared catalog before processing them.

base <reference>
Configures the base of the shared catalog.
prefix prefix: <reference>
Inserts a prefix mapping into the shared catalog.

Commands

Commands are executed interactively from the shell command loop. References are interpreted as a relative URI and resolved against the current base of the shared catalog before processing them.

base
Inspects the current base of the shared catalog.
prefix prefix:
Inspects the mapping for the namespace prefix in the shared catalog.
prefixes
Inspects prefix namespace mappings in the shared catalog.
decode <reference>
Decodes the input origin identified by reference into the working graph.
encode <reference>
Encodes the working graph into the output target identified by reference.
clear base
Restores the base of the shared catalog to its default value, that is the file: URI for the current working directory.
clear prefix prefix:
Removes the mapping for the namespace prefix from the shared catalog.
clear prefixes
Restores prefix mappings for the shared catalog to their default state, that is to the following set of well-known namespaces: RDF, RDFS, OWL, XSD.
clear graph
Empties the working graph.
clear
Clears base and prefix mappings for the shared catalog and empties the working graph.
system "command"
Executes a system command relaying standard output and error streams to the console.
time command
Times the execution of a shell command .
table command
Reports bindings returned by command as a TSV table.
trace
Inspects the stack trace of the last untrapped exception.
quit
Terminates the shell.
