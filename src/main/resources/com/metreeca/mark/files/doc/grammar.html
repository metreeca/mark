Introduction

Work in progress. Details may change in future releases.

This reference defines the formal grammar for the human-readable syntax of the Datalog-like language supported by the Oroboro Java RDF processing framework.

Keywords are case insensitive. Comments and whitespaces between tokens are ignored.
